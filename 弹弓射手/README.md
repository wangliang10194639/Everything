# 弹弓奇兵：城堡冲刺

一款创新的微信弹弓射击小游戏，融合了经典的弹弓玩法与Roguelike元素。

## 🎮 游戏特色

- **创新玩法**：单指拖拽弹弓发射英雄，精准打击敌人城堡
- **多样英雄**：5种独特英雄，每种都有专属技能和特性
- **策略收集**：符文系统，收集组合激活强大增益效果
- **渐进挑战**：动态关卡系统，难度逐步提升
- **商业化完整**：集成广告和内购系统，支持可持续运营

## 🏗️ 技术架构

### 核心系统
```
弹弓射手/
├── js/
│   ├── config/           # 游戏配置文件
│   ├── managers/         # 管理器系统
│   │   ├── AdManager.js     # 广告管理

│   ├── controllers/      # 控制器
│   │   └── SlingShotController.js  # 弹弓控制器
│   ├── characters/       # 角色系统
│   │   ├── Hero.js       # 英雄基类及实现
│   │   └── Enemy.js      # 敌人基类及实现
│   ├── scenes/           # 场景管理
│   │   └── GameScene.js  # 主游戏场景
│   ├── base/             # 基础类
│   ├── libs/             # 第三方库
│   ├── npc/              # NPC相关

│   ├── runtime/          # 运行时组件

├── images/               # 图片资源
├── audio/                # 音频资源
```

### 技术特点
- **模块化设计**：各系统独立开发，便于维护和扩展
- **事件驱动**：使用观察者模式实现系统间解耦
- **配置驱动**：游戏平衡性通过配置文件管理
- **微信原生**：充分利用微信小游戏API特性

## 🚀 快速开始

### 开发环境准备
1. 安装微信开发者工具
2. 确保Node.js环境（可选，用于构建脚本）

### 项目构建
```bash
# 给脚本添加执行权限
chmod +x build.sh check_compatibility.sh full_build_check.sh

# 运行完整构建检查（推荐）
./full_build_check.sh

# 或分别运行
./check_compatibility.sh  # 兼容性检查
./build.sh               # 构建项目
```

### 微信开发者工具导入
1. 运行构建脚本后，在项目根目录会生成 `dist/` 文件夹
2. 打开微信开发者工具
3. 选择"导入项目"，目录选择 `dist/`
4. 填入正确的AppID（可在project.config.json中配置）
5. 点击"预览"进行真机测试

## 🎯 核心玩法

### 基本操作
- **发射英雄**：在屏幕底部弹弓区域按住并拖拽，松手发射
- **力度控制**：拖拽距离决定发射力度
- **角度调整**：拖拽方向控制发射角度
- **特殊技能**：每个英雄都有独特技能，冷却期间可使用

### 游戏目标
- 发射英雄击毁敌人的城堡和防御设施
- 收集掉落的符文，激活组合效果
- 通关所有关卡，挑战更高分数

### 英雄系统
1. **石头战士** - 基础型，石爆技能
2. **火焰法师** - 高攻击，火风暴技能
3. **闪电忍者** - 高速度，连锁闪电技能
4. **治愈之水** - 辅助型，治疗光环技能
5. **冰霜射手** - 控制型，冻结射击技能

### 符文系统
- **攻击符文**：提升伤害、暴击率等
- **防御符文**：增加生命、减伤等
- **特殊符文**：金币加成、经验加成等
- **组合效果**：收集特定符文组合激活额外能力

## 💰 商业化设计

### 广告系统
- **激励视频**：复活、双倍奖励、符文重掷等
- **插屏广告**：关卡结束后展示
- **Banner广告**：主界面底部常驻（可购买去除）

### 内购系统
- **英雄解锁**：钻石购买新英雄
- **货币包**：金币和钻石充值包
- **月卡系统**：每日奖励和特权加成
- **去广告包**：一次性购买去除所有广告

## 🔧 开发指南

### 添加新英雄
1. 在 `js/config/gameConfig.js` 中添加英雄配置
2. 在 `js/characters/Hero.js` 中创建新的英雄类
3. 添加对应的图片资源
4. 在商店系统中注册新英雄

### 添加新敌人
1. 在 `js/config/gameConfig.js` 中添加敌人配置
2. 在 `js/characters/Enemy.js` 中创建新的敌人类
3. 添加对应的图片资源
4. 在关卡生成逻辑中加入新敌人类型

### 调整游戏平衡
修改 `js/config/gameConfig.js` 中的各项参数：
- 英雄属性（伤害、速度、生命值）
- 敌人强度（生命值、攻击力）
- 弹弓参数（力度系数、角度调整）
- 符文掉率和效果数值

## 🐛 常见问题

### 构建失败
- 确保所有必需文件都存在
- 检查文件权限设置
- 验证JSON配置文件格式正确

### 兼容性问题
- 运行 `check_compatibility.sh` 检查问题
- 确认微信基础库版本
- 避免使用过新的ES语法

### 性能问题
- 减少同时存在的特效数量
- 优化图片资源大小
- 合理使用requestAnimationFrame
