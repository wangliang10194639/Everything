# 微信开发者工具操作指南

## 🚀 快速开始

### 1. 打开项目
在微信开发者工具中：
- 点击 "小程序/小游戏" 
- 选择 "导入项目"
- 项目目录选择：`/root/Everything/弹弓射手`
- AppID填写：`wx82dcba8bb3bf5cef`
- 项目名称：弹弓射手

### 2. 必要的预处理步骤

#### 清除缓存（重要！）
```
工具菜单 → 清除缓存 → 清除所有缓存
```

#### 重新编译
```
点击工具栏上的"编译"按钮
```

### 3. 预览和调试

#### 模拟器预览
- 编译成功后自动在模拟器中显示
- 可以测试基本功能

#### 真机预览
```
点击"预览"按钮 → 生成二维码 → 微信扫码测试
```

#### 真机调试
```
点击"真机调试"按钮 → 生成调试二维码 → 连接设备调试
```

## 🐛 常见问题处理

### 模块未定义错误
如果仍然出现模块路径错误：

1. **确认缓存已清除**
2. **检查导入语句格式**：
   ```javascript
   // 正确格式（当前采用）
   import GameScene from 'scenes/GameScene';
   import Main from 'js/main';
   ```

3. **查看控制台详细错误**：
   - 开发者工具 → 控制台面板
   - 仔细阅读错误堆栈信息

### 白屏问题
1. 检查 `game.js` 是否正确执行
2. 确认 `js/main.js` 中的初始化代码
3. 查看是否有JavaScript运行时错误

### 资源加载失败
1. 检查 `images/` 和 `audio/` 目录文件是否存在
2. 确认文件路径引用正确
3. 查看网络面板资源加载状态
