# 《弹弓奇兵：城堡冲刺》完整设计文档

## 1. 项目概述

### 1.1 项目信息
- **项目名称**：弹弓奇兵：城堡冲刺
- **游戏类型**：微信小游戏（混合玩法）
- **开发引擎**：Cocos Creator 2.4.11
- **目标平台**：微信小游戏
- **项目状态**：开发中

### 1.2 核心玩法
- **操作方式**：单指拖拽弹弓发射英雄
- **游戏机制**：弹弓射击 + Roguelike元素 + 轻策略收集
- **游戏时长**：单局3-5分钟
- **目标用户**：25-35岁上班族，碎片化时间娱乐

## 2. 技术架构

### 2.1 技术选型
```
前端框架：Cocos Creator 2.4.11
编程语言：JavaScript (ES6+)
物理引擎：Cocos2d-JS内置物理系统
音频系统：Cocos Audio Engine
数据存储：localStorage + 微信云存储
广告系统：微信广告SDK
```

### 2.3 核心系统设计

#### 2.3.1 游戏管理器系统
- **GameManager**：游戏主控制器，管理全局状态
- **LevelManager**：关卡管理，处理敌人生成和波次控制
- **AdManager**：广告系统，处理各种广告展示逻辑
- **ShopManager**：商店系统，管理商品购买和货币系统

#### 2.3.2 角色系统
- **Hero基类**：所有英雄的基础类，包含通用属性和方法
- **Enemy基类**：敌人基础类，处理受击和死亡逻辑
- **角色继承体系**：不同英雄类型继承Hero基类实现特有技能

#### 2.3.3 控制系统
- **SlingShotController**：弹弓控制器，处理拖拽、发射逻辑
- **物理系统集成**：利用Cocos物理引擎实现真实的抛物线运动
