# 弹弓射手发布准备清单

## 📋 发布前必做事项

### 1. 基础配置检查
- [ ] 确认 `project.config.json` 中的 AppID 已正确填写
- [ ] 检查 `game.json` 中的设备方向和基础配置
- [ ] 验证 `project.private.config.json` 中的个人配置

### 2. 广告配置
- [ ] 在微信公众平台申请广告位ID
- [ ] 替换代码中的占位符广告ID：
  - 激励视频：`adunit-rewarded-video-id`
  - 插屏广告：`adunit-interstitial-id`  
  - Banner广告：`adunit-banner-id`
- [ ] 测试广告功能是否正常工作

### 3. 支付配置（如需内购）
- [ ] 申请微信支付权限
- [ ] 配置支付回调地址
- [ ] 测试支付流程

### 4. 资源优化
- [ ] 压缩图片资源（建议使用 TinyPNG 等工具）
- [ ] 优化音频文件大小
- [ ] 删除开发用的调试代码和日志

### 5. 性能测试
- [ ] 在多款手机上测试流畅度
- [ ] 检查内存占用情况
- [ ] 验证长时间游戏的稳定性

## 📊 数据埋点建议

### 核心指标
```
// 用户行为埋点示例
track('game_start')        // 游戏开始
track('level_complete')    // 关卡完成
track('hero_purchase')     // 英雄购买
track('ad_show')          // 广告展示
track('ad_click')         // 广告点击
track('iap_success')      // 内购成功
```

### 关键数据维度
- 用户留存率（次日/7日/30日）
- 关卡通关分布
- ARPU（平均每用户收入）
- 广告eCPM
- 内购转化率
