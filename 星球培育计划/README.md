# 星球培育计划 - 微信小游戏

## 项目简介

《星球培育计划》是一款以宇宙星球为主题的合成+放置类微信小游戏。玩家通过收集陨石碎片，合成各种星球元素，逐步培育并装饰自己的星球，打造独一无二的宇宙家园。

## 核心玩法

### 1. 合成系统
- **基础合成**：3个相同等级的元素合成1个更高等级的元素
- **特殊合成**：特定组合可合成稀有元素（如：水+土壤=种子）
- **合成失败**：高级元素合成有失败概率，可通过广告提升成功率

### 2. 放置系统
- **资源产出**：宇宙基地自动收集原始陨石
- **离线收益**：根据离线时长计算产出，最多累计12小时
- **基地升级**：提高产出速度和存储上限

### 3. 图鉴系统
- **元素收集**：每合成一个新元素，点亮图鉴
- **收集加成**：图鉴收集提供永久性的离线收益加成
- **稀有度系统**：普通、稀有、罕见、史诗、传奇五个等级

### 4. 装饰系统
- **星球装扮**：将高级元素装饰到主星球上
- **装饰加成**：不同装饰提供不同的产出加成
- **个性化定制**：打造独一无二的星球外观

## 商业化设计

### 1. 广告变现

#### 激励视频广告
- **立即收获**：立刻获得4小时的离线产出
- **合成加速**：让下一次合成瞬间完成
- **双倍收益**：接下来30分钟内，所有产出翻倍
- **额外机会**：合成失败时，保留高级材料

#### 插屏广告
- **触发时机**：场景切换时（如从合成界面返回主基地）
- **频率控制**：每日最多5次，间隔不少于5分钟

#### Banner广告
- **位置**：界面底部非核心操作区域
- **显示规则**：仅在特定界面显示

### 2. 内购设计

#### 虚拟货币
- **游戏币**：基础货币，通过游戏获得，用于基础合成
- **钻石**：高级货币，主要通过广告或内购获得，用于高级功能

#### 内购商品
- **新手礼包**：少量钻石+游戏币+特殊元素
- **月卡**：每日领取钻石，特殊装饰权限
- **钻石包**：不同档位的钻石购买
- **限定装饰**：节日或特殊主题的装饰物品

## 性能优化建议

### 1. 资源优化
- 使用图片压缩工具减小资源体积
- 合理使用图片格式（WebP、PNG等）
- 实现资源懒加载，减少初始加载时间

### 2. 代码优化
- 使用代码压缩工具减小JS体积
- 避免频繁的DOM操作
- 合理使用事件委托，减少事件监听器数量

### 3. 内存管理
- 及时释放不再使用的对象
- 避免内存泄漏
- 合理使用定时器，避免不必要的性能消耗

## 常见问题解答

### 1. 游戏无法加载
- 检查网络连接是否正常
- 尝试清除微信缓存
- 确认微信版本是否为最新

### 2. 游戏数据丢失
- 检查是否登录了微信账号
- 尝试重新登录微信
- 联系客服提供游戏ID和问题描述

### 3. 广告无法观看
- 检查网络连接是否正常
- 确认微信广告功能是否正常
- 尝试重启游戏或微信
