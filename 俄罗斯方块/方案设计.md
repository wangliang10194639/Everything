# 俄罗斯方块网页版 - 方案设计文档

## 1. 项目概述

### 1.1 项目名称
俄罗斯方块网页版 (Tetris Web Version)

### 1.2 项目目标
开发一个基于HTML5 Canvas和JavaScript的经典俄罗斯方块游戏，支持本地浏览器直接打开运行，无需服务器环境。

### 1.3 技术栈
- **HTML5**: 游戏页面结构
- **CSS3**: 现代化UI设计和动画效果
- **JavaScript (ES6+)**: 游戏核心逻辑
- **Canvas API**: 游戏画面渲染

## 3. 核心功能设计

### 3.1 游戏棋盘
- **尺寸**: 10列 × 20行
- **方块大小**: 30像素 × 30像素
- **总尺寸**: 300像素 × 600像素

### 3.2 方块系统

#### 3.2.1 七种经典方块
1. **I型** (青色) - 4格直线
2. **J型** (蓝色) - L形短边在左
3. **L型** (橙色) - L形短边在右
4. **O型** (黄色) - 2×2正方形
5. **S型** (绿色) - Z形镜像
6. **T型** (紫色) - T字形
7. **Z型** (红色) - Z形

#### 3.2.2 方块属性
- 类型标识 (1-7)
- 形状矩阵 (4×4或更小)
- 当前位置 (x, y坐标)

### 3.3 游戏机制

#### 3.3.1 方块移动
- **左移**: ← 键
- **右移**: → 键
- **下落**: ↓ 键 (加速)
- **旋转**: ↑ 键
- **硬降**: 空格键 (直接落到底)

#### 3.3.2 碰撞检测
- 边界检测 (左右边界、底部边界)
- 重叠检测 (与已固定方块检测)

#### 3.3.3 行消除
- 检测满行（无空单元格）
- 消除满行并上方方块下移
- 一次消除多行获得额外分数

### 3.4 计分系统

#### 3.4.1 基础得分
| 消除行数 | 得分 |
|---------|------|
| 1行     | 100  |
| 2行     | 300  |
| 3行     | 500  |
| 4行     | 800  |

#### 3.4.2 等级系统
- 每消除10行升1级
- 等级提升增加下落速度
- 最高等级无限制

### 3.5 游戏状态

#### 3.5.1 状态类型
- **就绪**: 游戏加载完成，等待开始
- **进行中**: 正在游戏中
- **暂停**: 游戏暂停，可继续
- **结束**: 游戏结束，显示得分

#### 3.5.2 状态转换
```
就绪 → 开始 → 进行中 → 暂停 → 继续 → 进行中
                      ↓
                    结束 → 开始
```

### 4.1 类设计

class TetrisGame {
    constructor() {
        this.board = [];      // 棋盘数据
        this.currentPiece;    // 当前方块
        this.nextPiece;       // 下一个方块
        this.score = 0;       // 得分
        this.level = 1;       // 等级
        this.lines = 0;       // 消除行数
        this.isPaused = false;

    }
    
    // 核心方法

    start();          // 开始游戏

### 4.3 核心算法

#### 4.3.1 碰撞检测算法
```javascript
function collide(board, piece) {
    for (let row = 0; row < piece.shape.length; row++) {
        for (let col = 0; col < piece.shape[row].length; col++) {
            if (piece.shape[row][col]) {
                const newX = piece.x + col;
                const newY = piece.y + row;
                
                // 边界检测
                if (newX < 0 || newX >= COLS || newY >= ROWS) {
                    return true;
                }
                
                // 重叠检测
                if (newY >= 0 && board[newY][newX]) {
                    return true;
                }
            }
        }
    }
    return false;
}
```

#### 4.3.2 旋转算法
```javascript
function rotate(piece) {
    const N = piece.shape.length;
    const rotated = [];
    
    // 矩阵转置 + 水平翻转
    for (let i = 0; i < N; i++) {
        rotated[i] = [];
        for (let j = 0; j < N; j++) {
            rotated[i][j] = piece.shape[N - 1 - j][i];
        }
    }
    
    piece.shape = rotated;
}
```

#### 4.3.3 行消除算法
```javascript
function clearLines() {
    outer: for (let row = ROWS - 1; row >= 0; row--) {
        // 检查是否满行
        for (let col = 0; col < COLS; col++) {
            if (!board[row][col]) {
                continue outer;
            }
        }
        
        // 移除满行
        const line = board.splice(row, 1)[0].fill(0);
        board.unshift(line);
        row++; // 重新检查当前行
    }
}
```
