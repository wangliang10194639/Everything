## 1. 游戏简介

俄罗斯方块（Tetris）是一款经典的益智游戏，由苏联程序员阿列克谢·帕基特诺夫于1984年发明。游戏目标是通过移动和旋转从屏幕顶部下落的各种形状的方块，使它们在屏幕底部排列成完整的水平行，消除得分。

本版本是基于HTML5 Canvas和JavaScript开发的网页版游戏，无需安装任何软件，直接在浏览器中即可游玩。

### 1.1 游戏特色
- ✅ 纯前端实现，无需服务器
- ✅ 现代化UI设计，视觉效果出色
- ✅ 流畅的动画效果
- ✅ 响应式布局，适配不同屏幕
- ✅ 完整的计分和等级系统
- ✅ 支持暂停/继续功能

## 2. 快速开始

### 2.1 环境要求
- **操作系统**: Windows、macOS、Linux
- **浏览器**: Chrome、Firefox、Safari、Edge（最新版本）
- **网络**: 可选（用于加载字体，不影响游戏运行）

### 2.2 启动游戏

#### 方法一：直接打开文件
1. 找到项目文件夹中的 `index.html` 文件
2. 双击文件或右键选择"用浏览器打开"
3. 游戏将在默认浏览器中启动

#### 方法二：使用浏览器打开
1. 打开任意现代浏览器（Chrome、Firefox等）
2. 点击菜单 → 文件 → 打开文件
3. 导航到项目文件夹，选择 `index.html`
4. 点击"打开"按钮

#### 方法三：拖拽到浏览器
1. 打开浏览器窗口
2. 将 `index.html` 文件拖拽到浏览器窗口中
3. 游戏将自动加载

### 2.3 开始游戏
1. 点击页面上的 **"开始游戏"** 按钮
2. 游戏将立即开始，方块将从顶部下落
3. 使用键盘控制方块移动和旋转

## 3. 操作指南

### 3.1 键盘控制

| 按键 | 功能 | 说明 |
|------|------|------|
| ← | 左移 | 方块向左移动一格 |
| → | 右移 | 方块向右移动一格 |
| ↓ | 加速下落 | 方块加速向下移动 |
| ↑ | 旋转 | 顺时针旋转方块90度 |
| 空格 | 硬降 | 方块直接落到底部 |
| P | 暂停/继续 | 暂停或继续游戏 |
| Enter | 重新开始 | 游戏结束后按Enter重新开始 |

### 3.2 鼠标/触摸控制

当前版本主要支持键盘操作，鼠标点击功能如下：

- **开始游戏**: 点击"开始游戏"按钮
- **暂停游戏**: 点击"暂停"按钮
- **重新开始**: 点击"重新开始"按钮
- **游戏结束**: 点击"再来一局"按钮

### 3.3 操作技巧

#### 3.3.1 基础技巧
1. **提前规划**: 在方块下落时就要考虑下一个方块的放置位置
2. **保持平整**: 尽量保持棋盘表面平整，避免出现难以填补的空洞
3. **利用预览**: 观察"下一个方块"预览，提前规划策略

#### 3.3.2 进阶技巧
1. **快速消除**: 尝试一次消除4行（Tetris）获得最高分数
2. **等级提升**: 消除行数越多，等级越高，下落速度越快
3. **硬降使用**: 在紧急情况下使用空格键快速下落

## 4. 游戏规则

### 4.1 基本规则
1. 方块从屏幕顶部中央位置生成
2. 方块自动向下掉落
3. 玩家可以控制方块的移动和旋转
4. 方块落地后会被固定在当前位置
5. 当方块堆积到顶部时游戏结束

### 4.2 方块类型

游戏包含7种经典方块，每种方块有不同的形状和颜色：

| 方块 | 名称 | 颜色 | 形状 |
|------|------|------|------|
| I | I型 | 青色 | 4格直线 |
| J | J型 | 蓝色 | L形（短边左） |
| L | L型 | 橙色 | L形（短边右） |
| O | O型 | 黄色 | 2×2正方形 |
| S | S型 | 绿色 | Z形镜像 |
| T | T型 | 紫色 | T字形 |
| Z | Z型 | 红色 | Z形 |

### 4.3 消除规则
- 当一行完全被方块填满时，该行会被消除
- 消除行后，上方的所有方块会向下移动
- 一次可以消除1-4行
- 消除行数越多，得分越高

### 4.4 游戏结束条件
- 方块堆积到棋盘顶部
- 新生成的方块无法放置

## 5. 计分系统

### 5.1 基础得分

消除行数与得分的关系：

| 消除行数 | 基础得分 | 倍数 |
|---------|---------|------|
| 1行 | 100 | ×1 |
| 2行 | 300 | ×3 |
| 3行 | 500 | ×5 |
| 4行 | 800 | ×8 |

### 5.2 等级系统

#### 5.2.1 升级规则
- 每消除10行，等级提升1级
- 等级从1开始，无上限

#### 5.2.2 速度变化
- 等级越高，方块下落速度越快
- 基础下落间隔：1000毫秒
- 每升1级，下落间隔减少100毫秒
- 最低下落间隔：100毫秒

### 5.3 得分计算公式
```
最终得分 = 基础得分 × 当前等级
```

**示例**：
- 1级时消除4行：800 × 1 = 800分
- 5级时消除4行：800 × 5 = 4000分

## 6. 常见问题

### Q1: 游戏无法启动怎么办？
**A**: 请尝试以下步骤：
1. 确认使用现代浏览器（Chrome、Firefox、Safari、Edge）
2. 清除浏览器缓存
3. 尝试使用其他浏览器打开
4. 检查HTML文件是否完整

### Q2: 键盘按键无响应怎么办？
**A**: 请检查：
1. 游戏是否已暂停（按P键继续）
2. 浏览器窗口是否处于激活状态
3. 是否有其他程序占用了键盘
4. 尝试点击游戏画面后再操作

### Q3: 游戏画面显示异常怎么办？
**A**: 请尝试：
1. 刷新页面（按F5）
2. 调整浏览器缩放比例（Ctrl+0）
3. 更新浏览器到最新版本
4. 尝试使用无痕模式打开

### Q4: 如何暂停游戏？
**A**: 按键盘上的 **P** 键可以暂停/继续游戏，或点击页面上的"暂停"按钮。

### Q5: 游戏结束后如何重新开始？
**A**: 有两种方式：
1. 点击"再来一局"按钮
2. 按Enter键

### Q6: 可以自定义游戏设置吗？
**A**: 当前版本暂不支持自定义设置，后续版本可能会添加。

### Q7: 游戏支持手机/平板吗？
**A**: 当前版本主要针对桌面端优化，移动端可能需要外接键盘。如需移动端支持，请期待后续版本更新。

## 7. 技术信息

### 7.1 文件结构
```
俄罗斯方块/
├── index.html    # 主页面文件（HTML结构）
├── style.css     # 样式文件（CSS设计）
├── game.js       # 游戏逻辑（JavaScript）
├── 方案设计.md    # 方案设计文档
└── 使用说明.md    # 使用说明文档
```

### 7.2 技术栈
- **HTML5**: 页面结构
- **CSS3**: 样式设计、动画效果
- **JavaScript (ES6+)**: 游戏逻辑
- **Canvas API**: 游戏画面渲染

### 7.3 浏览器兼容性
| 浏览器 | 最低版本 | 推荐版本 |
|--------|---------|---------|
| Chrome | 60+ | 最新版 |
| Firefox | 55+ | 最新版 |
| Safari | 11+ | 最新版 |
| Edge | 79+ | 最新版 |

### 7.4 性能要求
- **CPU**: 任意现代处理器
- **内存**: 最低 256MB
- **显卡**: 无特殊要求（使用Canvas 2D渲染）

### 7.5 离线使用
本游戏完全支持离线使用：
- 所有资源都在本地
- 无需网络连接
- 无需服务器支持
