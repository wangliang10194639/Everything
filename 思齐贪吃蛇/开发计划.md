# 思齐贪吃蛇 - 精简版开发计划

### 任务2：核心类设计

**Snake.js**
```javascript
class Snake {
  constructor(x, y, length) {
    this.body = [{x, y}];
    this.direction = 'right';
    this.nextDirection = 'right';
    this.length = length;
    this.speed = 150;
    this.canWallPass = false;
  }
  
  move()              // 移动
  changeDirection(dir)  // 改变方向
  grow()              // 增长
  checkCollision(bounds)  // 碰撞检测
  draw(ctx)           // 绘制
}
```

**Food.js**
```javascript
class Food {
  constructor(type = 'normal') {
    this.type = type;  // normal, life
    this.x = 0;
    this.y = 0;
  }
  
  generate(snake)     // 生成
  draw(ctx)           // 绘制
}
```

**Game.js**
```javascript
class Game {
  constructor() {
    this.state = 'idle';  // idle, playing, paused, gameover
    this.score = 0;
    this.highScore = 0;
    this.lives = 3;
    this.snake = null;
    this.food = null;
  }
  
  init()              // 初始化
  start()             // 开始
  update()            // 更新
  draw()              // 绘制
  handleInput(input)  // 处理输入
}
```

### 任务3：经典模式机制

**生命系统：**
- 初始3条命
- 失误一次 -1条命
- 生命道具：每20分出现，恢复1条命（最高3条）

**难度递增：**
- 每10分速度提升5%
- 最高提升50%（150ms → 75ms）

**穿墙模式：**
- 达到30分后解锁
- 消耗5分可穿墙一次

const STORAGE_KEYS = {

class StorageManager {
  static get(key, defaultValue) {
    try {
      return wx.getStorageSync(key) || defaultValue;
    } catch (e) {
      return defaultValue;
    }
  }
  
  static set(key, value) {
    try {
      wx.setStorageSync(key, value);
    } catch (e) {
      console.error('存储失败:', e);
    }
  }
}
