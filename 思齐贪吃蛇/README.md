# 思齐贪吃蛇 - 微信小游戏

🎮 一款经典贪吃蛇微信小游戏，简洁易玩，视觉效果出色。

## 📱 游戏特色

- **经典模式**：3条命，难度递增，生命道具
- **精美视觉**：粒子特效、流畅动画、渐变色系
- **完美适配**：全面屏适配、触摸控制
- **数据保存**：最高分本地存储

## 🎯 游戏规则

1. **目标**：控制蛇吃到更多食物，获得更高分
2. **生命**：初始3条命，失误一次扣1条命
3. **难度**：每10分速度提升5%，最高提升50%
4. **生命道具**：每20分出现心形道具，恢复1条命
5. **穿墙模式**：达到30分后解锁，可消耗5分穿墙一次

## 🚀 快速开始

### 1. 安装微信开发者工具

下载并安装 [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)

### 2. 导入项目

1. 打开微信开发者工具
2. 点击"导入项目"
3. 选择"思齐贪吃蛇"文件夹
4. 输入AppID：`wxad8a414f26812641`（或使用测试号）
5. 点击"确定"

### 3. 运行游戏

1. 点击微信开发者工具中的"编译"按钮
2. 在模拟器中预览游戏
3. 或使用真机调试

## 🎮 操作说明

- **滑动控制**：上下左右滑动屏幕控制蛇的移动方向
- **点击暂停**：点击屏幕暂停/继续游戏
- **键盘控制**：使用方向键或WASD键控制

## 📊 数据存储

- **最高分**：本地存储，永久保存
- **游戏设置**：本地存储，包括音效、震动等设置
- **统计数据**：总游戏次数、总得分
