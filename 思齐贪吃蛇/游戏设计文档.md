## 2. 经典模式详细设计

### 2.1 游戏规则

**核心玩法：**
- 目标：在有限的生命次数内获得最高分
- 初始生命：3条命
- 吃到食物 +1分，失误一次 -1条命
- 生命耗尽则游戏结束，显示最终得分

### 2.2 特色机制

**生命道具：**
- 每20分出现一个生命恢复道具（红色心形）
- 拾取后恢复1条命，最高3条

**难度递增：**
- 每10分速度提升5%
- 最高提升50%（即速度从150ms降至75ms）

**穿墙模式：**
- 达到30分后解锁穿墙能力
- 消耗5分可穿墙一次
- 穿墙时蛇从屏幕一侧穿过，从另一侧出现

### 2.3 计分系统

- 普通食物：+1分
- 生命道具：+5分（恢复1条命）
- 穿墙技能：-5分/次

## 3. 视觉设计规范

### 3.1 配色方案

**主色调：**
- 背景色：深空蓝 `#0f172a`
- 网格线：海军蓝 `#1e293b`
- 蛇头：翡翠绿 `#10b981`
- 蛇身：青绿色 `#14b8a6`
- 普通食物：珊瑚红 `#f43f5e`
- 超级食物：金色 `#fbbf24`
- 道具：多彩渐变

**UI配色：**
- 主按钮：渐变紫 `#8b5cf6` → 粉红 `#ec4899`
- 次按钮：深蓝 `#3b82f6`
- 文字：白色 `#ffffff`
- 分数板：半透明黑 `#00000080`

### 3.2 动画效果

**蛇的移动：**
- 身体摆动动画：正弦波运动
- 吃到食物：缩放动画 + 粒子爆发
- 死亡动画：渐隐 + 碎片散开

**食物动画：**
- 浮动效果：上下轻微浮动
- 旋转效果：缓慢自转
- 出现/消失：缩放渐变

**UI动画：**
- 按钮点击：缩放反馈
- 分数变化：数字滚动效果
- 模式选择：卡片滑动

### 3.3 粒子系统

**食物吃掉效果：**
- 粒子数量：8-12个
- 粒子颜色：与食物相同
- 粒子寿命：0.5-1秒
- 扩散范围：50px

**爆炸效果：**
- 粒子数量：20-30个
- 粒子颜色：金色到橙色渐变
- 粒子寿命：1-1.5秒
- 扩散范围：100px

### 5.2 核心类设计

**Snake类（蛇）：**
```javascript
class Snake {
  constructor(x, y, length) {
    this.body = [{x, y}];  // 蛇身数组
    this.direction = 'right';  // 当前方向
    this.nextDirection = 'right';  // 下一个方向
    this.length = length;  // 目标长度
    this.speed = 150;  // 移动速度（毫秒）
    this.canWallPass = false;  // 是否可以穿墙
  }
  
  move()           // 移动蛇
  changeDirection(dir)  // 改变方向
  grow()           // 蛇身增长
  checkCollision()  // 检查碰撞
  draw(ctx)        // 绘制蛇
}
```
