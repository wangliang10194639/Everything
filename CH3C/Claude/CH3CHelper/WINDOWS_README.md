# 澄海3C辅助软件 - Windows版使用说明

## 系统要求

- **操作系统**: Windows 7 / 8 / 10 / 11
- **.NET Framework**: 3.5 或更高版本
- **Python**: 3.7+（如使用源码运行）
- **权限**: 管理员权限（必需）

## 快速开始

### 方式一：使用打包好的EXE文件（推荐）

1. **下载并解压**
   - 下载 `CH3CHelper.zip`
   - 解压到任意目录（如 `C:\CH3CHelper\`）

2. **以管理员身份运行**
   - 右键点击 `CH3CHelper.exe`
   - 选择"以管理员身份运行"
   - 点击"是"确认UAC提示

3. **启动魔兽争霸III**
   - 打开魔兽争霸III
   - 进入澄海3C地图

4. **使用辅助软件**
   - 在辅助软件界面输入命令
   - 例如：输入 `status` 查看状态

### 方式二：使用Python源码运行

1. **安装Python**
   - 下载 Python 3.7+：https://www.python.org/downloads/
   - 安装时勾选 "Add Python to PATH"

2. **安装依赖**
   ```cmd
   cd CH3CHelper
   pip install -r requirements.txt
   ```

3. **运行程序**
   - 双击 `run.bat`（自动以管理员身份）
   - 或手动运行：
     ```cmd
     cd CH3CHelper
     python main.py
     ```

### 方式三：自行打包EXE

1. **安装PyInstaller**
   ```cmd
   pip install pyinstaller
   ```

2. **运行打包脚本**
   - 双击 `build.bat`
   - 或手动运行：
     ```cmd
     cd CH3CHelper
     pyinstaller --clean CH3CHelper.spec
     ```

3. **获取EXE**
   - 打包好的文件在 `dist\` 目录下
   - 将整个 `dist\` 目录复制到其他电脑使用

## 配置文件说明

### 3C.txt 配置文件

程序使用 `configs\3C.txt` 配置文件，包含：

- **[normal]** - 基本信息（版本、描述等）
- **[orderdefine]** - 技能命令ID定义（200+技能）
- **[action]** - 动作脚本（884个脚本）
- **[hotkey.*]** - 英雄热键配置（每个英雄独立配置）
- **[物品]** - 物品配置
- **[技能.*]** - 技能详细配置
- **[商店.*]** - 商店物品配置

### 自定义配置

您可以编辑 `configs\3C.txt` 来：
- 添加新的热键
- 修改现有脚本
- 自定义英雄配置

修改后重启程序即可生效。

## 常见问题

### Q: 程序无法启动？
A: 请确保：
1. 以管理员身份运行
2. 安装了Python 3.7+（源码运行时）
3. 安装了所有依赖（源码运行时）

### Q: 找不到魔兽争霸III进程？
A: 请确保：
1. 魔兽争霸III正在运行
2. 使用了正确的魔兽版本（1.20e / 1.24b / 1.27）
3. 程序有管理员权限

### Q: 热键不生效？
A: 请检查：
1. 当前英雄是否正确
2. 热键配置是否存在
3. 程序是否在前台运行

### Q: 内存错误或崩溃？
A: 请确保：
1. 使用了正确版本的魔兽
2. 内存偏移配置正确
3. 不要在对战平台使用

### Q: 如何打包自己的EXE？
A: 运行 `build.bat`，或手动执行：
```cmd
pyinstaller --clean CH3CHelper.spec
```
