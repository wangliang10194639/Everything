# 澄海3C 游戏辅助软件 (CH3C Helper)

一款功能完整的魔兽争霸III澄海3C游戏辅助软件。

## 功能特性

### 核心功能
- **全图视野** - 移除战争迷雾，显示所有单位
- **英雄追踪** - 实时监控敌我英雄位置、状态、技能
- **热键管理** - 支持英雄专属热键、物品栏快捷键、按键替换
- **脚本引擎** - 执行自动化动作脚本
- **智能提示** - 敌人英雄出现、技能释放、低血量等警报

### 配置支持
- 完整支持3C.txt配置文件格式
- 200+ 技能定义
- 各英雄独立热键配置
- 自动连招脚本
- 物品购买配置

## 快速开始

### 安装依赖
```bash
cd CH3CHelper
pip install -r requirements.txt
```

### 运行程序
```bash
python main.py
```

### 交互命令
- `status` - 显示当前状态
- `map` - 切换全图功能
- `heroes` - 显示英雄列表
- `scripts` - 显示可用脚本
- `run <脚本名>` - 运行脚本
- `hero <英雄名>` - 切换英雄热键
- `quit` - 退出程序

## 模块说明

### 核心模块 (core/)

#### config_parser.py
解析3C.txt配置文件，支持：
- 版本与描述信息
- 技能命令定义
- 动作脚本
- 热键配置
- 物品配置

#### game_data.py
游戏数据结构定义：
- Unit, Hero, Building 等单位类型
- 技能与物品信息
- 游戏状态管理
- 英雄状态枚举

#### memory_reader.py
游戏内存读取：
- 进程检测与附加
- 内存读写操作
- 游戏状态更新
- 模拟数据模式（用于测试）

#### process_hook.py
进程钩子与输入模拟：
- 键盘模拟
- 鼠标模拟
- 热键监听
- 按键替换

### 功能模块 (modules/)

#### map_hack.py
全图视野功能：
- 移除战争迷雾
- 显示隐藏单位
- 小地图覆盖层

#### hero_tracker.py
英雄追踪系统：
- 实时位置追踪
- 技能释放检测
- 低血量警报
- 死亡通知

#### hotkey_manager.py
热键管理：
- 英雄专属热键
- 按键替换
- 按键屏蔽
- 物品栏快捷键

#### script_engine.py
脚本引擎：
- 动作脚本解析
- 条件判断
- 循环与跳转
- 自动执行

## 配置文件

3C.txt 包含以下配置节：
- `[normal]` - 基本信息
- `[orderdefine]` - 技能命令定义
- `[action]` - 动作脚本
- `[hotkey.*]` - 英雄热键
- `[物品]` - 物品配置
- `[技能.*]` - 技能配置
- `[商店.*]` - 商店配置

## 技术说明

### 内存操作
- 使用 pymem 库进行进程内存读写
- 支持真实模式与模拟模式
- 针对特定魔兽版本需要调整内存偏移

### 输入模拟
- 支持 pynput 和 keyboard 库
- 按键组合、按住、延时等操作
- 鼠标移动与点击

### 脚本语言
- 基于3C.txt原生命令格式
- 支持等待、跳转、条件判断
- 高级动作：等待出现、选择物体等

## 注意事项

⚠️ **重要提示**：
1. 本软件仅供学习和研究使用
2. 请勿在正式游戏或比赛中使用
3. 使用前请确保符合相关游戏规则
